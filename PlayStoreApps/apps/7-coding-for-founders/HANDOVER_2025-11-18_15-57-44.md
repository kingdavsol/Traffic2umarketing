## Deployment Guide

### Prerequisites
1. Node.js 18+ and npm
2. MongoDB Atlas account
3. Google Cloud credentials (OAuth)
4. Google AdMob account
5. Stripe account (optional)

### Environment Variables
```
NEXTAUTH_URL=https://app-domain.com
NEXTAUTH_SECRET=[generate-with-openssl]
DATABASE_URL=mongodb+srv://...
GOOGLE_ID=[from-google-cloud]
GOOGLE_SECRET=[from-google-cloud]
NEXT_PUBLIC_ADMOB_APP_ID=[from-admob]
```

### Build & Deploy
```bash
npm install
npm run build
npm run start
# Or deploy to Vercel/Netlify
vercel deploy
```

## Troubleshooting Guide

### MongoDB Connection Issues
1. Verify connection string is correct
2. Check IP whitelist in MongoDB Atlas
3. Ensure database user has proper permissions
4. Review `.env.local` configuration

### Google OAuth Failures
1. Verify redirect URI matches exactly
2. Check credentials in Google Cloud Console
3. Confirm NEXTAUTH_SECRET is set
4. Review NextAuth logs for details

### Ad Display Issues
1. Verify AdMob app ID in environment
2. Check ad unit IDs are configured
3. Ensure ads are not cached (test mode)
4. Review Google AdMob dashboard

### Slow Performance
1. Check database query performance
2. Enable caching layer (Redis)
3. Optimize images with Next.js Image
4. Review bundle size with `npm run analyze`
