# NeighborCash - Hyper-Local Community Rewards

Support local small businesses while earning cashback rewards.

## Features
- 🏬 Discover local merchants
- 💰 0.5-1% cashback from small businesses  
- 🤝 Community economic impact
- 📍 Location-based merchant discovery

## Tech Stack
- Next.js, React, Tailwind CSS
- SQLite database
- JWT authentication

## Getting Started
```bash
npm install
npm run dev
```

## Business Model
- Commission: 0.5-1% from merchants
- Premium listings: $14.99/month
- Local sponsorships and partnerships
