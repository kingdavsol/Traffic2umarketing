# EarnHub - Student Micro-Gig Platform

Flexible earning opportunities designed for students.

## Features
- 📚 Student-focused gigs (tutoring, homework help, research)
- 💵 Weekly payments
- ⏰ Flexible scheduling around school
- 🎓 School partnership programs
- 👥 Peer-to-peer learning

## Business Model
- Commission: 10-15% on gigs
- Premium profiles: $7.99/month
- School partnerships and sponsorships

## Tech Stack
- Next.js, React, SQLite
- JWT authentication
- Responsive design

## Getting Started
```bash
npm install && npm run dev
```
