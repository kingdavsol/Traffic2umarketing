# SkillTrade - Blue-Collar Gig Platform

Connect licensed trade professionals with homeowners needing quality work.

## Features
- 🔧 Job marketplace for plumbers, electricians, HVAC, etc.
- 💰 85% commission for contractors
- ⭐ Rating and review system
- 📱 Mobile-responsive design

## Tech Stack
- Next.js, React, Tailwind CSS
- Node.js API routes
- SQLite database

## Getting Started
```bash
npm install
npm run dev
```

Visit http://localhost:3000

## Business Model
- Commission: 15% on jobs
- Premium profiles: $9.99/month
- Advertising: Featured job listings
