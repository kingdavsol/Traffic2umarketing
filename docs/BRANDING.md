## Visual Identity

### Logo: QuickSell Monster

#### Design Concept
A cute, friendly monster character named the "QuickSell Monster" that represents:
- **Speed**: Energetic and quick
- **Friendliness**: Happy, approachable
- **Reliability**: Stable and dependable
- **Fun**: Playful and engaging

#### Character Design
```
Physical Characteristics:
- Round, friendly body shape
- Big happy eyes with sparkle
- Wide smiling mouth with tongue out
- Cute red/pink coloring
- Happy horns (not scary)
- Golden stars as achievements
- Friendly waves/hands
- Happy feet
```

#### Visual Elements
- **Head**: Circle shape, friendly proportions
- **Eyes**: Large, white with black pupils, sparkle in eyes
- **Eyebrows**: Happy/raised position
- **Mouth**: Big smile, playful tongue
- **Horns**: Small, cute, non-threatening
- **Body**: Elliptical, huggable shape
- **Belly Patch**: Lighter color for contrast
- **Limbs**: Short, cute, friendly
- **Stars**: Gold accent color showing achievements


### Typefaces

**Headings (H1, H2, H3)**

- Letter Spacing: -0.02em

- Font Weight: 400 (Regular)
- Font Size: 14-16px
- Line Height: 1.6
- Letter Spacing: 0

**Buttons**
- Font Weight: 600 (Semibold)
- Text Transform: Sentence case (not ALL CAPS)
- Line Height: 1.4

**Taglines**
- Font Weight: 500 (Medium)
- Font Style: Italic (optional)
- Letter Spacing: 0.02em

## Logo Variations

### 1. Full Logo (Primary)
**Usage**: Website header, app icon (padded), marketing materials
**Size**: Minimum 200px width
**File**: `logo.svg`

### 2. Icon Logo (Favicon)
**Usage**: Browser tab, app icon, favicons
**Size**: 32px - 512px
**File**: `favicon.svg`
**Background**: Solid color with rounded corners

### 3. Text + Icon (Horizontal)
**Usage**: Header branding, social media profiles
**Layout**: Icon on left, "QuickSell" text on right
**Spacing**: 16px between icon and text

### 4. Text Only
**Usage**: Social media, when space is limited
**Font**: Bold, sans-serif
**Color**: Brand blue (#007AFF)
**Style**: "QuickSell" in full, or "QS" as abbreviation

### 5. Icon Only (Monochrome)
**Usage**: Favicon, small icons, app shortcut
**Color**: Single color (preferably brand blue)
**Size**: Minimum 16px

## Logo Usage Guidelines

### Do's ‚úÖ

- Use the monster character in a friendly, positive context
- Maintain proper spacing (clear zone) around the logo
- Scale proportionally - never distort the logo
- Use approved color palette
- Place logo on contrasting backgrounds
- Use high-quality vector formats (SVG)

### Don'ts ‚ùå

- Don't rotate the logo (except for small icons)
- Don't change colors or use gradients
- Don't add drop shadows or complex effects
- Don't use in dark, scary contexts
- Don't stretch or compress proportions
- Don't combine with competing logos
- Don't use outdated versions

### Clear Space
Maintain clear space around the logo:
- **Minimum**: 20px on all sides (for web)
- **Icon only**: 10px minimum
- **No text overlay** on the logo itself

### Background Requirements
- **Logo on Blue**: Use white or light backgrounds
- **Logo on Pink**: Use white or dark backgrounds
- **Logo on White**: Logo remains vibrant
- **Logo on Dark**: Use with transparency or white outline

## Animation & Motion

### Recommended Animations

**Loading State**
- Gentle bounce: Monster bounces slightly
- Duration: 1-2 seconds per bounce
- Repeat: Continuous until loaded

**Success State**
- Celebrate: Monster raises arms in joy
- Stars appear around it
- Confetti effect (optional)
- Duration: 1-2 seconds

**Error State**
- Confused look: Monster tilts head
- Maybe sad face temporarily
- Then quickly returns to happy
- Helps users feel supported

**Interaction**
- Subtle scale on hover
- Blink occasionally (friendly)
- Follow cursor movement (advanced)
