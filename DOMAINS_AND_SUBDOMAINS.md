```
Record Type    Host              Value              TTL
────────────────────────────────────────────────────────

CNAME          *.traffic2u.com   traffic2u.com      3600
```

```
...
(repeat for all 68 subdomains)
...
A Record       quicksell.monster                 YOUR_VPS_IP        3600
A Record       www.quicksell.monster             YOUR_VPS_IP        3600
```

This requires 70+ DNS records (tedious but works if wildcard is not allowed).

## HOW TO ADD TO HOSTINGER

1. **Login** to your Hostinger hPanel
2. **Go to**: Domains → Your Domain → Manage DNS Records
3. **Add Record**:
   - Type: A Record
   - Name: traffic2u.com (or subdomain)
   - Value: YOUR_VPS_IP (e.g., 185.145.130.45)
   - TTL: 3600
4. **Click**: Add

**For wildcard (easier)**:
- Name: `*` (just the asterisk)
- Value: YOUR_VPS_IP
- This makes ALL subdomains resolve to this IP

## PORT ASSIGNMENTS

Each app runs on different ports. Once deployed, Nginx reverse proxies them:

```
Port     Apps (Example)
────────────────────────────────────
3000     Next.js apps (caption-genius, gig-credit, earn-hub, etc.)
5000     React Native Express backends (snap-save, peri-flow, etc.)
5432     PostgreSQL database
6379     Redis cache
27017    MongoDB (if used)
8080     Monitoring dashboard
9001-9010 Insurance monorepo apps
9011-9018 Business tools monorepo apps
```

Nginx listens on 80/443 and routes traffic based on domain to the correct backend port.

## DNS PROPAGATION

After adding DNS records:
- **Immediate**: May work right away
- **Short term** (5-30 min): Most providers start resolving
- **Full propagation** (up to 48 hours): All DNS servers worldwide updated

You can check propagation at: https://dnschecker.org/

## RENEW/UPDATE DOMAINS

- **Renewal**: Hostinger renews automatically (you can disable this)
- **Expiration**: Domain will stop working if not renewed
- **Email alerts**: Hostinger sends renewal notices
- **Grace period**: Usually 30 days after expiration

## SSL CERTIFICATES

All domains need SSL certificates for HTTPS:

```bash
# Single command to get certs for all domains:
sudo certbot certonly --nginx \
  -d traffic2u.com \
  -d *.traffic2u.com \
  -d quicksell.monster \
  -d www.quicksell.monster
```

Let's Encrypt will issue wildcard certificate for `*.traffic2u.com` covering all 68 subdomains!
