# SkillMatch

🏆 Competitive Skill-Based Games Tournament

## Key Features

- ✅ Real-money skill game tournaments
- ✅ Email/SMS verification
- ✅ JWT authentication
- ✅ Google Mobile Ads integration
- ✅ User profiles & gamification
- ✅ Full backend API

## Tech Stack

### Frontend
- React Native with Expo
- React Navigation
- Zustand for state management
- Axios for API calls
- Google Mobile Ads
- Linear Gradient UI

### Backend
- Node.js with Express
- MongoDB
- JWT authentication
- Nodemailer for emails
- CORS enabled

## Installation

### Backend
```bash
cd backend
npm install
cp .env.example .env
npm run dev
```

### Frontend
```bash
cd frontend
npm install
npm start
```

## Environment Variables

See `.env.example` files in both frontend and backend directories.
