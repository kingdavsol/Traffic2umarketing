## Overview
WarmInbox is an email warmup service that gradually increases email sending volume to improve deliverability and avoid spam folders. Essential for sales teams and marketers starting cold email campaigns.

### 2. Warmup Engine
**Priority:** CRITICAL

1. Start with low volume (5-10 emails/day)

3. Send emails to other WarmInbox accounts (network effect)

5. Auto-reply to create conversation threads

### 6. Email Content Generation

Generate human-like emails:

**Using OpenAI:**
```typescript
async function generateWarmupEmail(context: {
  senderIndustry: string;
  recipientIndustry: string;
  previousThread?: string;
}) {
  const prompt = `Generate a brief, natural business email...`;

  const response = await openai.chat.completions.create({
    model: 'gpt-4',
    messages: [{ role: 'user', content: prompt }],
    max_tokens: 150,
  });

  return response.choices[0].message.content;
}
```

## 🔐 Security Considerations

**Critical:**
- Encrypt OAuth tokens at rest (use crypto library)
- Never log email credentials
- Secure token storage (separate encryption key)
- Token refresh mechanism
- Rate limiting on email sending
- Prevent abuse (max accounts per user)

**Privacy:**
- Never read user's actual emails (only warmup emails)
- Clear privacy policy
- GDPR compliance (EU users)
- Data deletion on account close
- Transparent about pool participation
