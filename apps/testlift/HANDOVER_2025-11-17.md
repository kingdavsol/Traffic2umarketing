## 🎯 Test Types to Support

1. **A/B Test:**
   - Two variants (A vs B)
   - Simple comparison
   - Most common

2. **Multivariate Test:**
   - Multiple elements tested simultaneously
   - Combinations (headline A + button B)
   - Requires more traffic

3. **Redirect Test:**
   - Compare completely different pages
   - Different URLs
   - Good for redesigns

4. **Split URL Test:**
   - Similar to redirect
   - Different paths

## 🔌 Technical Decisions

**Tracking Script:**
- Pure JavaScript (no dependencies)
- <10KB minified + gzipped
- Supports IE11+ (or modern only?)

**DOM Changes:**
- MutationObserver for reliability
- RequestAnimationFrame for performance
- Fallback for old browsers

**Event Storage:**
- PostgreSQL (Prisma) for aggregates
- ClickHouse or BigQuery for raw events (optional, scalability)

**CDN:**
- Cloudflare for tracker.js
- Edge caching
- Global distribution
