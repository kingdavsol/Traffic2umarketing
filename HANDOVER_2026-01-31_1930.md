## Issues Fixed

### 1. Marketplace Signup URLs Not Showing (BulkMarketplaceSignup.tsx)

**Problem:** The `MARKETPLACE_SIGNUP_URLS` object used title case keys ('eBay', 'Facebook') but `marketplace.id` is lowercase ('ebay', 'facebook'), causing the signup links to never appear.

**Fix:** Changed all keys to lowercase and added more marketplaces:
```typescript
// BEFORE (BROKEN):
const MARKETPLACE_SIGNUP_URLS: Record<string, string> = {
  'eBay': 'https://signup.ebay.com/pa/crte',
  'Facebook': 'https://www.facebook.com/marketplace',
  // ...
};

// AFTER (FIXED):
const MARKETPLACE_SIGNUP_URLS: Record<string, string> = {
  'ebay': 'https://signup.ebay.com/pa/crte',
  'facebook': 'https://www.facebook.com/marketplace/create/item',
  'craigslist': 'https://accounts.craigslist.org/login',
  'mercari': 'https://www.mercari.com/signup/',
  'poshmark': 'https://poshmark.com/signup',
  'etsy': 'https://www.etsy.com/join',
  'depop': 'https://www.depop.com/signup/',
  'vinted': 'https://www.vinted.com/signup',
  'offerup': 'https://offerup.com/signup',
  'tiktok': 'https://seller-us.tiktok.com/account/welcome',
  'instagram': 'https://www.instagram.com',
  'nextdoor': 'https://nextdoor.com',
};
```

### 2. Copy/Paste Instructions Not Visible (CreateListing.tsx)

**Problem:** After creating a listing, users were immediately redirected to `/listings` (line 490: `setTimeout(() => navigate('/listings'), 2000)`), so they NEVER saw the step 2 "Publish Results" page with marketplace hyperlinks and copy/paste instructions.

**Fix:** Modified `handleSubmit` to show step 2 with copy/paste data instead of redirecting:
- Set `publishResults` with manualPosts data
- Call `setActiveStep(2)` to show copy/paste instructions
- Keep background publishing for automated marketplaces
- Only redirect when no marketplaces selected

### 3. Missing Marketplace URLs (CreateListing.tsx)

**Problem:** `getMarketplaceUrl` function only had 3 URLs (Facebook, OfferUp, Mercari), causing "Open in Browser" buttons to not work for most marketplaces.

**Fix:** Added all 12 marketplace URLs:
```typescript
const getMarketplaceUrl = (marketplace: string) => {
  const urls: {[key: string]: string} = {
    facebook: 'https://www.facebook.com/marketplace/create/item',
    offerup: 'https://offerup.com/sell/',
    mercari: 'https://www.mercari.com/sell/',
    ebay: 'https://www.ebay.com/sl/prelist/suggest',
    craigslist: 'https://post.craigslist.org/',
    poshmark: 'https://poshmark.com/create-listing',
    etsy: 'https://www.etsy.com/your/shops/me/tools/listings/create',
    depop: 'https://www.depop.com/products/create/',
    vinted: 'https://www.vinted.com/items/new',
    tiktok: 'https://seller-us.tiktok.com/product/manage',
    instagram: 'https://www.instagram.com',
    nextdoor: 'https://nextdoor.com/sell/',
  };
  return urls[marketplace.toLowerCase()] || '#';
};
```
