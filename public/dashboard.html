API Integration

Submit Metrics

POST /api/metrics

{
  "appName": "your-app-name",
  "displayName": "Your App Display Name",
  "metrics": {
    "users": 1000,
    "downloads": 50,
    "revenue": 100.50,
    "adImpressions": 5000,
    "adClicks": 250,
    "sessions": 2000
  }
}

Example cURL

curl -X POST https://monitor.sourcevida.com/api/metrics \
  -H "Content-Type: application/json" \
  -d '{
    "appName": "test-app",
    "displayName": "Test App",
    "metrics": {"users": 100, "downloads": 5, "revenue": 50.00}
  }'
