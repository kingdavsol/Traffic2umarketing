API Information

Submit Metrics

POST to: /api/metrics

{
  "appName": "your-app-name",
  "displayName": "Your App Display Name",
  "date": "2024-01-15",
  "metrics": {
    "users": 1000,
    "activeUsers": 500,
    "downloads": 50,
    "revenue": 100.50,
    "adImpressions": 5000,
    "adClicks": 250,
    "sessions": 2000,
    "custom": {
      "custom_metric": "value"
    }
  }
}

Register App

POST to: /api/apps

{
  "appName": "your-app-name",
  "displayName": "Your App Display Name"
}

Get App Stats

GET: /api/apps/:appId/summary

Example cURL Command

curl -X POST http://your-server/api/metrics \
  -H "Content-Type: application/json" \
  -d '{
    "appName": "bizbuys",
    "displayName": "BizBuys",
    "metrics": {
      "users": 1500,
      "downloads": 75,
      "revenue": 250.00
    }
  }'
